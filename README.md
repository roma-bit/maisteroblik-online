# Майстер Облік — онлайн кошторис для майстрів

Безкоштовний онлайн сервіс для будівельних майстрів України.

## Для кого
- Маляр
- Електрик
- Сантехнік
- Плиточник
- Штукатур
- Гіпсокартонщик

## Що робить
- Онлайн кошторис по роботах і матеріалах
- Облік кількох об'єктів одночасно
- Фіксація авансів, чеків і залишку до оплати
- PDF для клієнта за 1 клік
- Працює з телефону
